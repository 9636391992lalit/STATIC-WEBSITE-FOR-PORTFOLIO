# Developer Portfolio - Static Site

This repository contains a responsive static developer portfolio built with HTML, CSS, and JavaScript.

Quick start:

1. Open `index.html` in your browser.

Files:

- `index.html` — main site
- `css/styles.css` — styles
- `js/script.js` — small interactive behaviors

Notes:

- The contact form is a placeholder and does not send messages to a backend.
- Customize text, projects, and styles to match the developer's profile.
